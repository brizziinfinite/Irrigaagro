## Decisões técnicas

- Rs em MJ/m²/dia (unidade NASA) → W/m² via `÷ 0.0864` para passar ao `calcETo` existente
- `evapoPlugfield` no `WeatherDay` interface (opcional, `undefined` para fontes não-Plugfield) — sem quebrar callers existentes
- `rs_source` default `'unknown'` para rows antigas; novas rows terão `'nasa'` ou `'plugfield_fallback'`
- Super admin identificado no cliente via `NEXT_PUBLIC_SUPER_ADMIN_EMAILS` (simples, sem backend extra)
