## Decisões técnicas

- Light mode descartado: tema dark é o padrão de qualidade para o produto; light exigiria redesenho completo dos tokens (gradientes, glows, sombras — todos pensados para escuro)
- CSS vars mantidas mesmo sem light mode: reduz acoplamento visual, facilita manutenção futura
- `isMobile` local com `matchMedia` listener: padrão mais limpo que `useWindowWidth` hook externo
