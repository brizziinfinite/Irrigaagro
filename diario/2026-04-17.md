## Decisões técnicas

- `rowSpan` só funciona corretamente em `tbody` — não usar com `thead` separado
- Logo definitivo usa **lowercase** no subtítulo: `textTransform:uppercase` com `letterSpacing` distorce proporções
- `document.title` como técnica de controle de nome do PDF — simples, sem dependências, funciona em todos os browsers
- `visibility: hidden` no `@media print` em vez de `display: none` para suprimir sidebar/header — mais compatível
- Skill de logotipo criada como fonte única de verdade para evitar retrabalho em sessões futuras

## Problemas resolvidos

- Logo subtítulo mais largo que o wordmark → lowercase + letterSpacing mínimo
- PDF imprimia sidebar e header → `visibility: hidden` global + `visibility: visible` no `.print-only`
- Nome do arquivo PDF incorreto → manipulação de `document.title` antes de `window.print()`
- `rowSpan` quebrando layout → mover tudo para `tbody`, sem `thead`
- `react-leaflet@5` quebrando build do AgroManager → downgrade para `4.2.1`
